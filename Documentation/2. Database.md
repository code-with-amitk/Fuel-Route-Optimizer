## Database and fuel station model

### FuelStation Table (Model)
- Fuel prices are given in [fuel-prices-for-be-assessment.csv](../fuel-prices-for-be-assessment.csv) but it does not have no latitude/longitude. We need coordinates to match stations to points along the route. We will get (lat, long) of truckstop offline and store in Database, to save time for online query time.
- FuelStations would be stored inside FuelStation Table
