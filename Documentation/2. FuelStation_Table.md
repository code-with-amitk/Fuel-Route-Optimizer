#### Understanding [manage.py](../manage.py)
- This is Django's command-line utility for administrative tasks which is a entry script that boots Django, and Django then treats `import_fuel_prices` as a management command name resolved from a Python file under `management/commands/`
- manage.py sets `DJANGO_SETTINGS_MODULE` and calls `django.core.management.execute_from_command_line(sys.argv)`


#### offline filling of fuel_optimizer_fuelstation table
0. Create a Command class `fuel_optimizer/management/commands/import_fuel_prices.py` which would be run as `python manage.py import_fuel_prices`
1. Inside `import_fuel_prices.py`, `Command` class inherits from `BaseCommand`, and Django calls its `handle()` method after parsing any arguments like `--csv, --clear, or --dry-run`. In file, those options are defined in `add_arguments()`, then `handle()` contains the actual import logic
2. This will 

- 2a. Parse the assessment read CSV once and pass the csv to for loop
- 2b. Create a HTTP request where input = f"{station.city}, {station.state}, USA" from csv file and send to "https://api.openrouteservice.org/geocode/search" to get lattitude, longitude
- 2c. FuelStation entry keep appending to geocoded_batch list
- 2d. Once geocoded_batch list > batch_size, its added to database
```py
class Command(BaseCommand):
    def handle(self, *args, **options):
        stations = self._parse_and_deduplicate(csv_path)    #2a
            for index, station in enumerate(stations, start=1):
                    response = requests.get(                            #2b
                        "https://api.openrouteservice.org/geocode/search",
                        headers={"Authorization": key},     //key from .env
                        params={
                            "text": f"{station.city}, {station.state}, USA",
                            "boundary.country": "USA",
                            "size": 1,
                        },
                        timeout=timeout,
                    )
                    lat, lng, confidence = response
                    
                    geocoded_batch.append(              #2c
                        FuelStation(
                            opis_id=station.opis_id,
                            name=station.name,
                            address=station.address,
                            city=station.city,
                            state=station.state,
                            rack_id=station.rack_id,
                            retail_price=station.retail_price,
                            latitude=lat,
                            longitude=lng,
                            geocode_confidence=confidence,
                        )
                    )

                    if len(geocoded_batch) >= batch_size:   #2d
                        inserted += self._bulk_insert(geocoded_batch, batch_size)
```
