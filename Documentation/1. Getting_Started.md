
## Getting started

### Install Django, Create Project
Check python version 3.10(works with Django 5.x).
```py
python3 --version && which python3
Python 3.10.12
/usr/bin/python3
```

Create python venv
```
python3 -m venv .venv && source .venv/bin/activate && python --version && which python && pip --version
Python 3.10.12
```

Install packages & create [reqiurements.txt](../requirements.txt) for python packages installed
```
cd /home/amit/Fuel-Route-Optimizer && source .venv/bin/activate && which python && which pip && 
python -m pip install --upgrade pip && 
python -m pip install "Django>=5.1,<5.2" djangorestframework django-environ requests
pip install "Django>=5.1,<5.2" djangorestframework django-environ requests
pip freeze | grep -E '^(Django|djangorestframework|django-environ|requests|asgiref|sqlparse|urllib3|certifi|charset|idna|typing_extensions)=' > requirements.txt 
```

Create Project
```
django-admin startproject fuel_route_api .
```

Create app fuel_optimizer, its standalone donot create inside fuel_route_api project
```
python manage.py startapp fuel_optimizer
```

Run migration
```
python manage.py migrate
```

Start django server
```
cd /home/amit/Fuel-Route-Optimizer && source .venv/bin/activate && python manage.py runserver 127.0.0.1:8000
```
