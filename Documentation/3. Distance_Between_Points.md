## Distance between point1(lon1,lat1), point2(lon2,lat2)

### Understanding [manage.py](../manage.py)
- This is Django's command-line utility for administrative tasks which is a entry script that boots Django, and Django then treats `fuel_optimizer/management/commands/test_ors_clients` as a management command name resolved from a Python file under `fuel_optmizer/management/commands/`
- manage.py sets `DJANGO_SETTINGS_MODULE` and calls `django.core.management.execute_from_command_line(sys.argv)`
