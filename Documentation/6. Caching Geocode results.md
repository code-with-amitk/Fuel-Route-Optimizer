

## Caching

cache ORS responses in Django’s cache framework(django.core.cache.backends.locmem.LocMemCache)

**Caching `services/ors_cache.py`**

| Cache key | Stores | TTL |
|---|---|---|
| `ors:geocode:{sha256(address)}` | lat, lng, label, confidence | 24 h (configurable) |
| `ors:route:{start_lat,lng}:{finish_lat,lng}` | GeoJSON geometry, distance, duration | 24 h |

- Present **`LocMemCache`** in dev (`settings.CACHES`)
- Production: swap to Redis via `CACHE_URL`

if address is not cached, cache it else return Cached address
```
mock_geocode.return_value = GeocodeResult(41.88, -87.66, "Chicago, IL, USA")
cached_geocode_address("Chicago, IL")
cached_geocode_address("Chicago, IL")

def cached_geocode_address(text: str, **kwargs) -> tuple[GeocodeResult, bool]:
    """Return geocode result and whether it came from cache."""
    key = geocode_cache_key(text)
    cached = cache.get(key)
    if cached is not None:
        return _deserialize_geocode(cached), True

    result = geocode_address(text, **kwargs)
    cache.set(key, _serialize_geocode(result), cache_ttl_seconds())
    return result, False
```
